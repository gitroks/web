# React 19 + TypeScript + Vite 7.3.1

This is a modern React application bootstrapped with Vite 7.3.1, using React 19, React Router v7.13, TypeScript and strict linting policies.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

### Lint

Run ESLint to check for code quality issues:

```bash
npm run lint
```

### Preview

Preview the production build:

```bash
npm run preview
```

This project uses strict ESLint rules including:
- No `any` types
- Explicit return types
- Strict boolean expressions
- No unused variables
- Prefer nullish coalescing and optional chaining
- And many more...
